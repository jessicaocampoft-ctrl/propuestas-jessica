# Guía para Administrar tus Propuestas

Ahora tu proyecto está organizado en **carpetas independientes** para cada cliente. Esto te permite tener propuestas personalizadas sin mezclar la información.

## 📂 Tu Nueva Estructura
```text
/ (Tu carpeta principal)
│
├── 📁 fighterx/  --> Propuesta para FighterX
│   └── index.html
│
├── 📁 kingbulltrainingcomisione/ --> Copia lista para editar
│   └── index.html
│
└── 📁 kingbulltrainingnocomisiones/ --> Copia lista para editar
    └── index.html
```

## 🚀 Cómo editar las nuevas propuestas
1. Entra a la carpeta de la propuesta que quieras modificar (ej: `kingbulltrainingcomisione`).
2. Abre el archivo `index.html`.
3. Edita los textos, precios y cambia el logo en la carpeta `fotos` de esa versión.

## 🌐 Cómo subirlo a GitHub (Actualizado)
Al tener carpetas, la forma más fácil de subir todo es:

1. Ve a tu repositorio en GitHub.
2. Si ya tenías archivos sueltos (`index.html`) en la página principal del repositorio, **bórralos** para mantener el orden.
3. Arrastra las **carpetas completas** (`fighterx`, `kingbull...`, etc.) hacia la ventana de GitHub.
   - *Nota: Si GitHub web no te deja arrastrar carpetas enteras fácilmente, tendrás que crear la carpeta usando "Add file -> Create new file" (escribiendo `carpeta/nombre.txt`) o usar la aplicación GitHub Desktop.*
   - **Método recomendado para principiantes:** Usa **GitHub Desktop** instalándolo en tu PC, es mucho más fácil para manejar carpetas.

## 🔗 Tus Enlaces Finales
Una vez subido, tus propuestas quedarán en direcciones separadas automáticamente:
- `tu-usuario.github.io/repositorio/fighterx`
- `tu-usuario.github.io/repositorio/kingbulltrainingcomisione`
- `tu-usuario.github.io/repositorio/kingbulltrainingnocomisiones`

¡Así podrás enviar un link profesional y específico a cada cliente!
